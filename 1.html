HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the
standard markup for creating web
pages and web applications. Web browsers receive
HTML documents from a web server of from local
storage and render them into multimedia web
pages. HTML describes the structure of a web
page semanitically and originally included cues for
the appearance of the document.
